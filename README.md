# Scoop

An unofficial PowerShell module wrapper for Scoop using PowerShell best
practices.

## Installing Scoop

> NOTE: Prerequisite [install scoop](https://scoop.sh/).

```powershell
Install-PSResource -Name Scoop
```

## Cmdlets

- `Find-ScoopApp`
- `Get-ScoopApp`
- `Install-ScoopApp`
- `Uninstall-ScoopApp`
- `Update-ScoopApp`
- `Get-ScoopBucket`
- `Register-ScoopBucket`
- `Unregister-ScoopBucket`
